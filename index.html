JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web. A diferencia de HTML y CSS, que se utilizan para estructurar y dar estilo al contenido, JavaScript agrega interactividad y dinamismo a las páginas. Algunos ejemplos de lo que puedes hacer con JavaScript incluyen actualizaciones de contenido en tiempo real, mapas interactivos, animaciones en 2D/3D y control de multimedia. Aunque es más conocido como un lenguaje de scripting para páginas web, JavaScript también se utiliza en otros entornos fuera del navegador, como Node.js, Apache CouchDB y Adobe Acrobat. Es un lenguaje basado en prototipos, multiparadigma, de un solo hilo y dinámico, con soporte para programación orientada a objetos, imperativa y declarativa.
